# 🌸 CycleAI

**Your cycle is unique. Your tracker should be too.**

CycleAI is an AI-powered menstrual health platform designed for irregular cycle tracking, prediction, and personalized insights.

---

## 🚀 Features

- 🧠 AI-based cycle prediction (ARIMA + ML)
- 📊 Cycle Health Score (0–100)
- 📅 Smart calendar tracking
- 💡 Personalized insights
- 🔒 Privacy-first architecture
- 🧾 Doctor sharing & reports

---

## 🏗️ Tech Stack

- Frontend: React + Tailwind
- Backend: Flask + JWT
- Database: PostgreSQL (Supabase)
- ML: Scikit-learn, Statsmodels
- Deployment: Render + Netlify

---

## ⚙️ Setup

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
flask db upgrade
flask run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
